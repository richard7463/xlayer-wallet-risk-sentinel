# Project Positioning

X Layer Wallet Risk Sentinel is a reusable pre-trade risk skill for X Layer agents.

## Core Claim

Agents should not receive execution rights without wallet-aware risk judgment.

## What The Skill Does

- reads wallet posture
- reads proposed trade size
- checks reserve stress, concentration, and impact
- returns one final verdict and a safe size cap

## Prize Strategy

Primary target:

- `Skills Arena`

Most relevant special-prize angle:

- `Best data analyst`
